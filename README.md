🍴 Student Tiffin Service Finder

![3957701](https://github.com/user-attachments/assets/55413b72-cad9-49eb-b2cb-72a4a84cc2bf)

 
A web-based platform designed to help students of GLA University, Mathura and nearby areas find reliable and affordable tiffin services in one place. The platform ensures convenience, trust, and transparency with an admin-managed system.

🚀 Why This Project?

Students living outside hostels often struggle to find good-quality and affordable meals. Information about tiffin services is scattered and unverified, making it hard for students to choose the right option.
This project provides a centralized platform where students can explore, compare, and save tiffin services with confidence.

🎯 Features

🔑 User Authentication – Secure login/signup with role-based access (Student, Owner, Admin)

🍲 Tiffin Listings – Service name, food type (Veg/Non-Veg), monthly price, timings, and contact details

🔍 Search & Filter – By food type, budget, and timings

⭐ Favorites/Bookmarks – Students can save preferred tiffin services

🛠️ Admin Panel – Admin verifies and manages listings

📱 Responsive UI – Works seamlessly on both desktop and mobile

🛠️ Tech Stack

Frontend → React / HTML, CSS, JavaScript

Backend → Node.js + Express

Database → MongoDB / MySQL

Authentication → JWT (JSON Web Token)

🗄️ Database Design

👤 Users → id, name, email, password, role (student/owner/admin)

🍴 Tiffin Services → id, name, type (veg/non-veg), price, timings, contact

⭐ Favorites → user_id, service_id

👨‍💻 Team Roles (4 Members)

🎨 Frontend Developer – UI design, search/filter, API integration

⚙️ Backend Developer – APIs, authentication, admin panel

🗂️ Database Manager – Schema design, queries, sample data

🧪 Tester/Integrator – Module integration, bug fixing, final presentation

✅ Expected Outcome

A fully functional Tiffin Service Finder platform for students.

Owners can add and manage services, students can search and bookmark options, and admins ensure authenticity.

Provides a real-life solution to make student life outside hostels more convenient.

✨ This project focuses only on Tiffin Services as the first phase (MVP), with potential for future expansion.
